# 实验八 基于FPGA的分频器设计

## 资源文件介绍

本仓库提供了一个资源文件，标题为“实验八 基于FPGA的分频器设计”。该资源文件主要包含以下两个扩展任务：

### 扩展任务1
使用开关控制蜂鸣器的状态，具体功能如下：
- 开关控制蜂鸣器不响。
- 开关控制蜂鸣器以1kHz频率响。
- 开关控制蜂鸣器以500Hz频率响。

### 扩展任务2
实现蜂鸣器发出“滴滴滴”的声音效果。
